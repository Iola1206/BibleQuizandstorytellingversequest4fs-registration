Import Live Data from Google Sheets

Paste your Google Apps Script Web App URL below and click Import. The dashboard will fetch all current registrations from your Google Sheet and display them here.

How to keep the dashboard in sync

1. Every time someone submits the registration form, a row is automatically added to your Google Sheet.
2. Come to this page and click Import from Sheets to pull all the latest registrations into the dashboard.
3. The dashboard will then show updated stats, charts and the full registrations table.
4. You can also add entries manually using the Add Entry page for anyone who registers via WhatsApp or phone.

Tip: Bookmark this dashboard page and visit it before every planning session to get the latest data.
